Class PoseUpdater

  • All Implemented Interfaces:
    
public class PoseUpdater

    This is the PoseUpdater class. This class handles getting pose data from the localizer and returning the information in a useful way to the Follower.

      • resetOffset

         void resetOffset()

        This resets all offsets set to the PoseUpdater. If you have reset your pose using the setCurrentPoseUsingOffset(Pose2d set) method, then your pose will be returned to what the PoseUpdater thinks your pose would be, not the pose you reset to.

      • getPose
